### 简介
一套面向开发者提供数字孪生的低代码开源工具包，实现基于cesium、echarts、earthsdk的可视化场景编辑、数据可视化编辑等功能，并且可以扩展监控视频接入、大模型接入、视频AI分析、业务应用低代码平台、物理网中台等功能，目前基于该开源工程在智慧城市、社区、园区、医院、景区已落地大量项目。

### 注意
git下载时，请下载master分支使用。
node > 14
### 开发模式

1. npm install
2. npm run dev
3. 浏览器地址栏输入 localhost:9529

### 打包

1. npm install
2. npm run build
